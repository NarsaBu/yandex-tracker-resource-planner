# Resource Planner App

## Назначение проекта

Данный проект создан для решения следующих проблем:
1. Определение задач, которые были завершены и у них была завышена оценка
2. Определение задач, которые вышли за рамки своей оценки по каким-либо причинам (неправильно оценили задачу, не учли какие то проблемы или списание времени было проведено неккоректно)
3. Определение, к какому сроку будет выполнен какой-либо проект с учетом капасити (количество человек опредленной роли на проекте)

## Как запустить проект?

### Запуск через IDE
1. Сделать mvn clean install
2. Заполнить .env файл
3. Запустить проект и пользоваться

### Запуск через Jar
1. Сделать mvn clean package
2. Вытащить архив [resource-planner-0.0.1.jar](target/resource-planner-0.0.1.jar) из target директории и переложить к себе в папку
3. Создать в этой папке с приложением файл external.properties
4. Прописать следующие переменные окружения в файле external.properties:
   HTTP_PORT=31660
   ROOT_LOG_LEVEL=INFO
   YANDEX_TRACKER_HOST=https://api.tracker.yandex.net/
   YANDEX_TRACKER_TOKEN=*сгеренируй свой токен. Инструкция ниже*
   YANDEX_TRACKER_X_ORG_ID=7867633
5. Запустить приложение используя следующий скрипт:
   MacOS/Linux ```java -jar resource-planner-0.0.1.jar '--spring.config.import=optional:file:./external.properties'```
   Windows: ```java -jar resource-planner-0.0.1.jar "--spring.config.import=optional:file:./external.properties"```

Путь до swagger: http://localhost:31660/swagger-ui/index.html

## Как сгенерировать токен

Чтобы получить токен:

1. Перейдите по ссылке https://oauth.yandex.ru. 
2. На странице Ваши приложения нажмите  Создать. 
3. В открывшемся окне выберите вариант Для доступа к API или отладки и нажмите Перейти к созданию. 
4. Укажите название приложения и почту для связи. 
5. Добавьте разрешения для доступа к данным пользователя. Чтобы выбрать разрешение, начните вводить его название в поле Название доступа:
6. Запись в трекер (tracker:write) — все операции с данными: создание, удаление, редактирование. 
7. Чтение из трекера (tracker:read) — только чтение данных. 
8. Нажмите Создать приложение. 
9. В личном кабинете Яндекс OAuth выберите созданное ранее приложение и скопируйте его идентификатор из поля ClientID. 
10. Сформируйте ссылку для запроса токена: https://oauth.yandex.ru/authorize?response_type=token&client_id=<идентификатор_приложения>
11. Войдите в аккаунт, от имени которого вы будете работать с API, и перейдите по сформированной ссылке. На странице появится последовательность символов — это OAuth-токен. Скопируйте его и сохраните. Укажите как значение в переменной окружения YANDEX_TRACKER_TOKEN
